How to edit Evocreo save data

The variables are decoded fine (albeit with some "question marks" that appeared between the data regularly that I HAVE NO IDEA WHAT IT IS FOR LOL.) This is really annoying, but i found ONE VULNERABILITY. which is the saved creos data on the EvoCo computer. You probably now know where this is going. If you're looking for Player Data, give it up.
