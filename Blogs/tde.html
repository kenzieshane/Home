journey of installing TDE (Trinity desktop Environment)

  To quote the official website itself,

  "The Trinity Desktop Environment (TDE) project is a computer desktop environment for Unix-like operating systems with a primary goal of retaining the function and form of traditional desktop computers [...] The TDE project began as a continuation of the K Desktop Environment (KDE) version 3. The name Trinity was chosen partly because the word means "three" and TDE was a continuation of KDE 3. TDE now is its own computer desktop environment project. The TDE project was founded by Timothy Pearson. Timothy is an experienced and skilled software developer and was the KDE 3.x coordinator of previous Kubuntu releases."

The journey

So, I've tried a little bit of different linux distributions... I've installed Fedora, Ubuntu, Linux MInt, KDE NEON and Garuda Linux.

I'm currently staying in KDE NEON because I'm just can't be bothered to reinstall the system.

I found the default kde plasma desktop environment is really heavy and slow for me. So, I searched in Google of what other Desktop Environments that is lightweight and found two that i thing is neat: Moksha and TDE

Moksha is okay, but in my experience the .desktop files that is accessed through the desktop would always ask to select an application to open them... I would use the dolphin file manager for that purpose, which can execute those programs(?)... Executing it through thunar works fine though, it only happened when executing .desktop files through the desktop... weird. So, i went out to use TDE, which suprisingly works very well! /fin
