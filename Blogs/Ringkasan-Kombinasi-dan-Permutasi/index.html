Summary of Permutations and Combinations

Friday, 29th November 2024
Author: Kenzie Shane S.

  (i) NOTICE: I use this amazing software to enter mathematical formulas into this website: Visual Math Editor by David Grima

Usage

Combination is used in the context of questions that do not pay attention to the order when taking/choosing something from the source, Permutation is used in the context of questions that require a sequence (usually marked by questions that ask 'how many (number of) ways are available to achieve something, etc') because the answer obtained without order will always be different from the one that is obtained. So thus Permutation is needed to sort that out instead of just using Combination.

Combination formula

nCr = n ! r ! ( n − r ) !

Permutation formula

nPr = n ! ( n − r ) !
